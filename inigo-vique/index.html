Plug-ins para Sublime

All Autocomplete
Ni idea
AutoFileName
Sirve para rellenar las rutas de ficheros cuando se ponen src= o href=
Color Picker
Ctrl+Shift+C, aparece un colorpicker y permite elegir un color. Lo añade a las CSS como #123456 directamente
Emmet
La maravilla al usar el Tab. Ayuda a escribir código HTML y CSS más rápido
HTML CSS JS Prettify
Permite hacer que el código se tabule correctamente para que sea más legible. Botón derecho -> HTML... -> Prettify Code. Mejor que no lo haga al guardar, que es lioso
Sublime Server
Crea un servidor HTML desde Sublime para no tener que usar las rutas file://. Para ponerlo en marcha sólo hay que pulsar Ctrl+Shift+P y escribir "Start Sublime Server"
