Cosas NPM

Http-server
npm install -g http-server
Crea un servidor http en la ruta en la que estamos poniendo el comando http-server en PowerShell. Es un servidor pequeño para hacer pruebas y no tiene demasiado problema para funcionar. Unar maravilla :).

Cosas para Chrome

WhatFont
Sirve para poder ver las fuentes que tiene una página
MeasureIt
Mide la distancia entre bloques. Es una regla.
Web Developer
Permite deshabilitar contenido para probar cómo funcionaría la web sin CSS, JS, etc...
Accessibility Developer Tools
Añade al F12 un sistema para poder comprobar si la web es accesible.
Colorzilla
Es un colorPicker que permite seleccionar colores de una web para saber su valor. ¡CUIDADO! Añade clases al body cargándose las que trae por defecto.
