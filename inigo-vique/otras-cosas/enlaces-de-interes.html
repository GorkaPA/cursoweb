Enlaces de interés

Fontawesome
Página con una fuente especial para iconos. Trae el listado de iconos, referencias y cómo linkarlos.
W3Schools
Teoría sobre páginas web, HTML, CSS y JS. Está bien para encontrar referencias sobre etiquetas HTML o propiedades CSS.
HTML Entities
Listado de entidades de HTML para añadirlas en vez de los simbolitos. Queda más limpio.
HTML Escape
Sirve para escapar contenido HTML y que pueda ser insertado dentro de etiquetas "pre" y "code" sin que sea leído
Appear in
Página para emitir pantalla. Es gratis hasta cierto número de espectadores
Can I Use
Sirve para saber qué propiedades están disponibles para determinados navegadores.
Dive into HTML5
Libro sobre los cambios a HTML5.
Google WebMaster Tools
Información de cómo mejorar el SEO de tu web para Google. Ayuda para las búsquedas de Google.
Pixabay
Web con imágenes libres de derechos para usar en nuestras webs.
RegExr
Generador de RegEx para validaciones de datos
CSSZenGarden
Página en la que el HTML es el mismo pero las CSS se pueden modificar para estilos completamente diferentes.
Codepen
Web para escribir HTML, CSS y Javascript y probarlo en timpo real
Unicode Table
Tabla con TODOS los caracteres Unicode. Absolutamente TODOS.

Octuweb

IconFinder
