En las descargas no cogemos un Modernizr estándar, creamos uno con las cosas que necesitemos. Siempre es adecuado añadir la opción html5shiv, que ayuda a los navegadores antiguos a reconocer las nuevas etiquetas de HTML5;

El script está a palo seco en el head. Comprueba si el navegador es capaz de utilizar "input required" y CSS grid. CSS grid peta en los Explorers y Edge, por ejemplo. El otro... Habrá que probarlo.

Al pulsar este botón muestra las características que el navegador no soporta, con unos bonitos mensajes de error en la consola.

Modernizr también nos da Polyfills, maneras de sustituir los bloques si estos no funcionan en determinados navegadores.
