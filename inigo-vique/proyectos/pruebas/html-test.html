Un poco de HTML

Encabezados

En HTML existen 6 niveles de encabezados

Por ejemplo, con esta etiqueta <h1>Título principal</h1> ponemos un encabezado de nivel 1

Listas

Ejemplo de lista ordenada

	
<ol>
  <li>Test1</li>
  <li>Test2</li>
  <li>Test3</li>
</ol>

Ejemplo de una lista desordenada

	
<ul>
  <li>Test1</li>
  <li>Test2</li>
  <li>Test3</li>
</ul>
