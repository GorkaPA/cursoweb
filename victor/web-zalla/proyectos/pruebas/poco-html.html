Un poco de HTML

Encabezados

En HTML existen 6 niveles de encabezados

Por ejemplo, con la etiqueta <h1>Título principal</h1> obtendremos un encabezado de nivel 1

Listas

Ejemplo de lista ordenada;

    
<ol>
	<li>lorem</li>
	<li>item2</li>
	<li>item3</li>
</ol>

Ejemplo de lista desordenada;

    
<ul>
	<li>lorem</li>
	<li>item2</li>
	<li>item3</li>
</ul>
