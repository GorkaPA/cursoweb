Apuntes CSS

Vincular los documentos HTML y CSS

Se puede de 3 formas distintas, pero solo una cumple las reglas de accesibilidad.

Esa forma correcta es crear un archivo .CSS con todos los estilos a aplicar al documento HTML y vincularlo dentro del head(al final a ser posible) con la etiqueta LINK

Elementos

existen 2 tipos de elementos. Elementos de bloque que no comparten linea ej h1, h2.. li, etc y elementos de linea que tienden a ocupar el mayor espacio disponible en la anchura de la pagina uno detras de otro ej. imagenes y inculos.

Dentro de los elementos de linea existen otros 2 subgrupos "inline block" e "inline"

los inline no soportan propiedades de css como la anchura width o altura height ni tampoco el padding-top y bottom ni el margin-top ni el bottom, para no disturbar el texto en lineas superiores o inferiores.

Selectores

Hay muchos tipos de selectores los mas basicos son los de etiqueta

tambien tenemos selectores por clase o id

A los selectores de clase siempre se les pone un punto delante para indicar que es una clase

mediante la "," se pueden agrupar selectores ej. h1,h2{.....}

Selector universal. es el asterisco y selecciona a todas las etiquetas de la pagina. se suele utilizar para resetear los margenes y padding de las cajas o para resetear valores por defecto del navegador. ES muy necesario practicamente obligatorio resetear a 0 los margenes y paddings de todas las cajas del documento html al principio de nuestro archivo de estilos.

Propiedades

Propiedades de texto

TODAS SE HEREDAN

color{} como su nombre indica es el color en ingles red, blue, green

font-size{} tamaño en pixels o % de la letra

font-weigth{} grosor de la letra normal sin negrita y bold con negrita

font-family{} especifica el tipo de letra de un parrafo puediendo darle diferentes opciones en el caso de que las primeras no puedan cargarse. Esta propiedad se hereda de padres a hijos en la etiquetas HTML. ej. si le añadimos a la propia etiqueta HTML una font-family times new roman, toda la pagina tendra ese estilo de letra

font-style{} normal es sin cursiva, italic con cursiva

font-variant{} es para poner las mayusculas a la misma altura que las minusculas, solo puede tomar el valor small-caps

text-transform. uppercase convierte texto a mayusculas. lowecase minusculas, capitalize primera letra mayuscula.

text-indent sirve para generar una sangria en un elemento que contenga texto

line-height o interlineado. distancia entre lineas.

background-color. color de fondo

text-decoration. sirve para poner subrayado con underline o quitarlo con none

letter-spacing y word-spacing distancia entre letras y palabras en pixeles.

Propiedades de caja

NO SE HEREDAN

Border : grosor estilo color. ej 3px solid green

lo mismo se utiliza para cada uno de los bordes por separado. border-top border-bottom, border-left, border-right

para quitar el borde se utiliza none

padding. Se refiere al relleno interior dentro de la caja un borde interior. Tambien se pueden definir cada uno de los lados

margin sirve para controlar la distacia entre cajas. y tmb se puede referenciar cada lado por separado.

width limita la ancura de un elemento

height limita la altura de un elemento

Display cambia la naturaleza de un elemento ej. de block a inline o a inline block

centrar elementos

elementos tipo bloque

le damos un margin left y un margin right con valor auto y una anchura para que o ocupe todo el ancho disponible

para centrar elementos de tipo linea utilizamos la propiedad text-align que se la aplicamos al contenedor del texto.

Pseudoclases

sirven para representar estados de elementos. Y son mas especificos que el mismo selector sin pseudoclase

link Se usa muy poco y aplica el estado de reposo a un vinculo.

visitedSe utiliza cuando el destino del vinculo ha sido visitado.

hoverSe aplica al estado de pasar el cursor del raton por encima de un elemento no tiene porke solo ser un vinculo.

activeSe aplica al estilo durante el tiempo que el usuario mantiene pulsado el boton del raton. Se usa muy poco.

focus Se puede aplicar a etiquetas y/o a controles o campos de formulario. El estilo visual del foco es normalmente un rectangulo alrrededor del elemento, eso se crea con la propiedad outline. Esta propiedad toma los mismos valores que la propiedad border, pero solo se puede definir el mismo estilo para todos sus lados. Tampoco aumenta el tamaño de la caja. Para anularlo se puede poner none o 0 pero deberiamos resaltar el foco de alguna manera.

Selectores compuestos, pseudoclases y descendientes

selectores descendientes :

Se usan para dar estilos a elementos que estan a cualkier nivel dentro de otro elemento referenciando por id o clase el elemento padre. Se utlizan 2 o mas selectores.

ej. #menu p {color:blue;} Daria color azul a todos los vinculos dentro de #menu

selector hijo :

Se utiliza el simbolo de mayor > y hace referencia a aquellos elementos que estan justo un nivel por debajo.

selector hermano:

se utiliza el simbolo ~ (alt+126) y hace referencia a todos los elementos que esten despues y al mismo nivel

ej. img~p{} hace referencia a todos los parrafos que esten despues de una imagen

selector hermano adyacente:

Se utiliza el simbolo + y sirve para referenciar a un elemento justo despues en el codigo de otro.

ej. h1+p{} hace referencia a un parrafo justo despues del elemento h1 en el codigo

selectores pseudoclases

first-child Selecciona al primer hijo. Si ponemos una etiqueta delante selecciona al primer hijo que sea del tipo de esa etiqueta

last-child lo mismo que el first-child pero para el ultimo

nth-child tambien le `podemos poner una etiqueta delante y le podemos pasar 3 tipos de parametros entre parentesis:

Un valor numericoseleccionara dentro de los hijos aquel que cumpla con la posicion dada.

odd o evenSeleccionara todos los pares o impares

una formula del tipo an+ba es el numero de pasos a dar en la lista de elementos, n es un valor que el navegador va cambiando secuencialmente desde 0 en adelante y b es la posicion de partida dentro de la lista de elementos

first-of-typeSelecciona el primer elemento del tipo de una etiqueta que se le especifique delante. Le da igual el nivel de herencia

last-of-typeLo mismo que el anterior pero para el ultimo

nth-of-type lo mismo que el nth-child, pero no esta limitado a los niveles de herencia

css para tablas

border-collapse

Sirve para combinar los bordes derecho e izquierdo de 2 celdas. el mas grueso prevalece

border-spacing

Define la distancia entre celdas y se aplica a la tabla entera. A las celdas no se les puede dar un margen. Esta propiedad es la unica manera de dar espacio entre celdas

empty-cells

Puede tomar 2 valores: show y hide y gestiona las celdas vacias.
