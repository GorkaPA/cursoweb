Listas

hay 3 tipos de listas: ordenadas, desordenadas y de definición

ul para las desordenadas y ol para las ordenadas. la etiqueta li sirve para cada elemento de la lista

las listas de definicion o listas semanticas estan formadas por 3 etiquetas dl para abrir y cerrar y otras 2 etiquetas para cada elemento:
dt para el término o clase o concepto y dd para la definicion, descripcion o instancia

anidacion de listas. Solo se pueden anidar listas ordenadas y o desordenadas, y siempre dentro de los li de las listas.

Imagenes

los navegadores solo soportan mapas de bits e imagenes vectorizadas dentro de los formatos de mapas de bits se soportan JPG, GIF, PNG y dentro de las vectorizadas SVG

Para insertar imagenes se usa la etiqueta img la cual tiene 2 atributos obligatorios src que es la ruta relativa a la imagen y alt que es el texto alternativo que saldra en vez de la imagen cuando no consigue cargarse dicha imagen

Vínvulos / links

La etiqueta para vinculos es a que tiene un atributo obligatorio href que indica el destino del vinculo. Puede ser un sitio externo al proyecto, a una pagina dentro de mi proyecto o incluso a otra parte de la misma pagina en la que estamos. tambien tiene atributos opcionales como target que hace que la pagina nueva se abra en otra pestaña. el unico valor que puede tomar target es _blank el atributo target esta totalmente desaconsejado para vinculos que redirigan a paginas del propio proyecto.

Los vinculos que redirigen a otras partes de la pagina necesitan que exista una barra de scroll, y que la etiqueta tenga un atributo id al que haremos referencia desde el href del vinculo. Siempre pondremos # antes del nombre de la id al hacerla referencia desde el href del vinculo.

Etiquetas para citas

hay 3 la q que se utiliza siempre dentro de una etiqueta que contiene texto para cuando solo un texto es la cita. blockquote que se utiliza para cuando todo un bloque de texto incluidas etiquetas y parrafos enteros es una cita.

Y para citar algo que no sean personas, como un pelicula, un libro, una obra etc... se utiliza la etiqueta cite

etiquetas para indices

existen 2 la etiqueta sub que es para poner subindices ej. H2O

Y la etiqueta sup que es para superindices ej . 22

etiquetas para abreviaturas y acronimos

se utiliza la etiqueta abbr para todo lo que sea abreviaturas o acronimos. ej. NASA

Etiqueta small

Al usarla por defecto el navegador hace el texto muy pequeño. Pero se suele utilizar para todo lo que va en letra pequeña como el copyright, o aclaraciones de superindices de palabras de un texto. O precios con asterisco para poner descripciones a pie de pagina

Etiquetas ins y del

Se utilizan para hacer modificaciones en el texto. Si queremos añadir informacion a un articulo pasado un tiempo se utiliza ins y para indicar algo incorrecto usamos del. Las 2 indican al navagador que el texto ha sido modificado y le aplican un estilo al texto modificado.

Etiqueta s

Se utliza para exponer algo que no es relevante pero no deseamos eliminar el texto. ej un precio rebajado

Etiqueta pre

Etiqueta que se utiliza para decirle al navegador que respete el formato del texto que contiene.

Etiqueta code

Se utiliza para poner codigo preformateado en el navegador y se utliza siempre jundo a la etiqueta 

<pre>

Etiqueta span

etiqueta auxiliar no semantica de tipo linea que sirve para rodear texto en concreto para darle estilo a unas palabras o trozos de texto dentro de un parrafo u otra etiqueta de texto

Etiqueta div

Muy utilizada

Etiqueta auxiliar no semantica de tipo bloque que sire para rodear a otras etiquetas para asignarles estilos.

Tablas

Se utilizan para mostrar informacion que este relacionada entre si en filas y columnas. ej un horario.

Tienen 3 etiquetas obligatorias:

tablepara abrirla y cerrarla

tr table row

th para las deldas de encabezados

td para las celdas de datos

rowspan y colspan

son 2 atributos que indican que una celda ocupa mas de una fila o columna cimbinando las celdas. colspan las combina en horizontal y rowspan en vertical

caption etiqueta que se usa para poner el titulo de la tabla que se pone inmediatamente despues de la etiqueta table

etiquetas estructurales de tabla

tbody, thead y tfoot los encabezados se suelen poner en thead, los datos en tbody y los resultados finales o sumatoris en tfoot

Etiquetas propias de HTML5

Todas van dentro del body y nos sirven para estructuras el documento(layout)

header

a al ppio del body y normalmente contiene el h1 obligatorio en todas las webs.

nav

Almacena un menu de la pagina. Ya sea el menu ppal o alguno secundario.

main

Almacena la parte de la pagina que no es comun para todas las paginas de la aplicacion.

section

Van dentro del main y engloban una seccion del mismo. Cada seccion debe de tener un header asociado.

footer

Va debajo del main y engloba el pie de pagina

formularios

Se utilizan para introducir datos

Se abren con la etiqueta form. Que tiene 2 atributos principales.

method que se usa para poner el metodo de envio del formulario ej. post

action que se usa para poner el destinatario de los datos del formulario

para cada elemto del formulario se utilizan estas etiquetas:

label se utiliza para el titulo de cada componente del formulario y se le suele poner una un atributo for para referenciar al elemento de cual es el titulo

input Se usa para introducir un tipo de dato. y tiene atributos importantes:

type Se usa para especificar el tipo de dato que recoge el input ej text, number, submit, date

namedeberia de ser igual al atributo for del label para que cuando pinchemos en el label ponga el focus en el input.

id Se usa como identificador y deberia tener el mismo valor que el name y el for del label

Para validar formularios se suele utilizar la etiqueta pattern junto con expresiones regulares

required es un atributo de los input que especifica que es un campo requerido

Etiqueta fieldset :

Se utiliza para englobar campos de un formulario relacionados

Se suele utilizar junto a la etiqueta legend que como su nombre indica es la leyenda que describe la relacion entre los campos
